### About

I'm a Backend Software Engineer with **3+ years** building distributed, event-driven systems in **Node.js** and **Go** across **AWS · Azure · GCP · OCI · AliCloud**.

At [**Turbot**](https://turbot.com) I work on **Guardrails** (preventive cloud governance) and contribute to all four of their open-source products: [Steampipe](https://github.com/turbot/steampipe), [Powerpipe](https://github.com/turbot/powerpipe), [Flowpipe](https://github.com/turbot/flowpipe), and [Tailpipe](https://github.com/turbot/tailpipe) — together holding **9,300+ GitHub stars**. Specialties: AWS SDK migrations, Azure Event Grid, PostgreSQL query optimization, and CIS / NIST / HIPAA / PCI compliance modules.

Recently I've been having fun with **MCP** (Model Context Protocol), real-time multiplayer game servers in Go (Nakama), and AI-agent frameworks on **Pydantic-AI**.
